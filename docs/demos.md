# Demos

## Landing Page Demos

The landing page (`index.html`) runs these demos directly:

- `Toggle Card`: scalar-driven card scale with intentional overshoot.
- `Number Spring`: scalar smoothing for counters and progress values.
- `Mouse Follower`: Vector2 pointer following with spring motion.

Global controls at the top tune spring parameters across all landing demos.

## Standalone Demo Folders

- `demos/moving-box`
- `demos/toggle-card`
- `demos/number-spring`
- `demos/mouse-follower`

## Run

```bash
npm run demo
```

Open `http://localhost:4173`.
